# AKuCU_lab2 Арифметическое сжатие


Входные данные: Файл в формате txt с текстом на русском или английском языке + режим работы (кодирование 
или декодирование).<br />
Реализовать алгоритм арифметического сжатия, можно опираться на алгоритм приведенный в книге Ватолина и 
Ратушняка из первой лекции.<br />
Программа должна уметь выполнять КОДИРОВАНИЕ и ДЕКОДИРОВАНИЕ – т.е. работать в двух режимах отдельно.<br />
Сжатый файл по объему должен быть меньше исходного.<br />
После декодирования файл должен совпадать с исходным файлом.<br />
Код должен быть «чистым».<br />
Необходимо реализовать алгоритм на целых числах.<br />
Таблицу частот хранить в закодированном файле.<br />
Выводить после кодирования степень сжатия и затраченное время.<br />
После декодирования выводить затраченное время.<br />
